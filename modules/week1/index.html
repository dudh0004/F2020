# 3. History of macOS

  • Apple first introduced the Macintosh in January, 1984. It cost $2495 USD, which is around $6000 USD in 2018.
  • It ran Mac OS (as known as System Software and now referred as Classic Mac OS) as its operating system, and was one of the first computers to incorporate a mouse.
  • The first Macintosh computers used versions 1 ‚Äì 9 of Mac OS.
  • Version 10 was a big change for the Mac.
  • The original operating system wasn‚Äôt built to handle modern software or hardware and it needed replacement.
  • Apple bought NeXT software for their operating system(NeXTSTEP). It became Mac OS X.
  • It is a version of UNIX, which has been around since 1970. It is mainly used in academia, and telecom companies. It is meant for expert users and is not user-friendly.
  • Apple Mac OS X added a graphical user interface to simplify using the operating system. It is still a UNIX operating system, but most of the details are now hidden from you.
  • On June, 2016 Apple renamed the Mac OS X to macOS.

# Selecting items in macOS Finder

Click on a file or folder to select it. Now hold the command key and select another item. This selects only those item you clicked while holding the command key.

Click on an item, and then hold the shift key and select another item. It selects every item between those items you clicked on.

Right-clicking brings up a context menu. If you have a two button mouse, click the right mouse button. If you have a single button mouse, hold the ctrl key and click.

You can also tag an item with colors. Select an item and right-click on it to bring up the context menu. Select one of the colors to tag those items. On the Finder window, select that color to see all your items.

# The root user

As a safety precaution, when you are logged in as a normal user, there are certain parts of the operating system that you can not access. You can not move or delete parts of the operating system, other programs, or other users‚Äô files.

To access important parts of the operating system, you must act as an administrator, or root user. UNIX and Linux computers call this the superuser.

In macOS, you use the command sudo. Try running sudo ls. You are running ls as the superuser. We will learn more about this later.

TIP

Your home directory is ~your-name but the root user home directory is /var/root
