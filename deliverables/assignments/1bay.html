# Assignment A1

How to answer and submit this assignment

You will create a file called last-first-a1.txt (e.g cezar-vladimir-a1.txt) that ONLY contains your answers.

This assignment contains 12 questions. Each question explicitly informs the number of lines expected as a answer.

For the 12 following questions you should have a file with 20 lines. If you don't know the answer to the question just add a blank line.

When you are finished, open Brightspace and go to the Activities > Assignments page.

Go to the A1 assignment. Upload your file.

WARNING

Your file should NOT contain anything else other than your answers. No question numbers, no names, no comments. Just the answers.
